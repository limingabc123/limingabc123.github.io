9. 常见问题排查

Q: 页面报错 “Unexpected identifier ‘s’”

  • 原因：JS 字符串中英文所有格 's 未转义，导致语法错误
  • 修复：将 assistant's 改为 assistant\'s

Q: GitHub Pages 返回 ERR_EMPTY_RESPONSE

  • 原因：GitHub Pages 正在构建中（push 后需 2-5 分钟）
  • 解决：等待几分钟后刷新，或清除浏览器缓存后重试

Q: 聊天无法发送消息

  • 检查项：
    1. Cloudflare Worker 是否在线（访问 Worker URL 测试）
    2. DeepSeek API Key 是否正确设置
    3. 浏览器 Console 是否有 CORS 错误
    4. Worker 的 DEEPSEEK_API_KEY 环境变量是否配置

Q: Cloudflare Worker 部署失败

  • 确认项：
    1. wrangler login 是否成功
    2. workers.dev 子域名是否已注册
    3. wrangler.toml 配置是否正确
    4. 是否有临时预览账号限制（需正式部署）
