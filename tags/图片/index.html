图片添加实战教程

本教程将一步步演示如何在 Hexo 博客中添加图片。

第一步：准备图片

  1. 将您的图片文件（如 my-photo.jpg）复制到以下目录之一：

    • source/_posts/images/ - 文章专用图片
    • themes/icarus/source/img/ - 主题通用图片

  2. 图片命名建议：

    • 使用英文小写字母
    • 使用连字符分隔单词
    • 示例：research-lab.jpg, coding-screenshot.png

第五步：故障排除

问题：图片不显示

  1. 检查路径：确认图片路径是否正确
  2. 检查文件名：注意大小写（Image.jpg 和 image.jpg 是不同的）
  3. 检查文件扩展名：确保扩展名正确（.jpg, .png, .gif）

问题：图片加载慢

  1. 压缩图片：使用 TinyPNG 压缩
  2. 调整尺寸：将大图调整为合适尺寸
  3. 使用 WebP 格式：现代浏览器支持，压缩率更高

问题：图片变形

总结

在 Hexo 中添加图片的关键步骤：

  1. 正确放置图片文件
  2. 使用正确的 Markdown 语法
  3. 优化图片以提高性能
  4. 添加有意义的替代文本
