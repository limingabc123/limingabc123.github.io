API 接口文档

// 发送消息到助手 POST http://localhost:8000/chat Content-Type: application/json { "message": "你好，请介绍一下李茗的研究方向", "history": [] } // 响应示例 { "reply": "李茗目前的研究方向主要包括...", "history": [...] }
# Python 客户端示例 import requests def chat_with_assistant(message, history=None): url = "http://localhost:8000/chat" data = { "message": message, "history": history or [] } response = requests.post(url, json=data) return response.json() # 使用示例 result = chat_with_assistant("你好") print(result["reply"])

📋 部署步骤

  1. 安装Ollama - 从官网下载并安装Ollama
  2. 下载模型 - 运行: ollama pull llama2
  3. 启动代理服务 - 运行: python ollama_proxy.py
  4. 测试连接 - 访问: http://localhost:8000/chat
  5. 集成到网站 - 在主页面中嵌入聊天组件
